# No Tutorial Button

A mod for Survive The Nights that disables the tutorial button in main menu.

You can build it using the STN Workshop and following the ModSDK guide https://docs.google.com/document/d/11ucs2otMwdD5a2-iJWKkRsPxZyUy8J7BrxeLhtFXvJg
